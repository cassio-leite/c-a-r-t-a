# Carta

Projeto de uma carta digital interativa desenvolvida com HTML e CSS puro.

## Descrição

O projeto "Carta" é uma página web estática que simula uma experiência de abertura de uma carta digital. Através de animações em CSS (efeito *hover*), o usuário pode interagir com o cartão para revelar uma mensagem personalizada.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da página.
- **CSS3**: Estilização, layout e animações (`transform`, `transition`, `perspective`, `hover`).
- **Google Fonts**: Tipografia utilizada (Roboto).

## Funcionalidades

- **Interatividade**: O cartão reage ao passar do mouse (*hover*), simulando o movimento de abrir uma carta.
- **Responsividade**: O layout possui ajustes básicos para diferentes tamanhos de tela.

## Como Visualizar

Este é um projeto estático. Para visualizar:

1. Clone o repositório.
2. Abra o arquivo `index.html` diretamente em qualquer navegador web.
